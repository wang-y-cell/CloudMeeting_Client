◆ mousePressEvent()

template<typename Base>
void FramelessWindow< Base >::mousePressEvent ( QMouseEvent * event)
inlineoverrideprotected

鼠标按下

处理优先级：

  1. 非左键 -> 交给基类
  2. 点在边缘热区 -> 进入缩放（grabMouse，防止缩小后丢事件）
  3. 点在标题栏且未点到按钮：
    • 已最大化 -> 按 x 相对比例还原后再拖
    • 普通状态 -> 记录拖动偏移并开始拖动
  4. 其它区域 -> 交给基类/子控件

是否允许最大化

为 false 时会隐藏最大化按钮，并禁用双击标题栏最大化。 登录窗口通常调用 setMaximizable(false)。
