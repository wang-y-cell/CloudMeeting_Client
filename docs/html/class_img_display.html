Detailed Description

将「图片来源 → 缩放/对齐策略 → 显示到控件」从业务逻辑里拆出， 便于在同一处切换绘制方式、绘制区域与刷新方式。

典型用法：绑定 QLabel，设置 DrawMode / 可选矩形区域，再调用 showImage()。
